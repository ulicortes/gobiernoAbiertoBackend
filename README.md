# Gobierno Abierto
## Frontend y backend del portal de Gobierno Abierto
- *Backend:*
- - Archivos:
Se va a usar Node para manejar el backend, Se puede enlistar los pdf de una determinada seccion y descargar un pdf en concreto.
Se deberia tener una carpeta con todos los pdf subidos, para usar su nombre como buscador.
En el metodo get de busqueda recibe seccion y nombre del archivo por parametro. Luego hace una busqueda ficticia y devuelve la ruta del archivo para finalmente realizar la descarga.
Esta carpeta deberia estar alojada en el mismo lugar donde corra la app asi se piede descargar sin problemas cada
uno de los archivos.
